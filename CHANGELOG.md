# Changelog

All notable changes to "ColorNest" will be documented in this file.

## [1.0.0] - 2026-03-29

### Added

- Initial release
- 12-color curated palette optimized for dark themes
- Automatic coloring on window open
- Slight color variations for Title Bar, Activity Bar, and Status Bar
- Random variation to ensure uniqueness across windows
- `Window Colors: Shuffle Color` command
- `Window Colors: Reset to Default` command
- Cross-platform support (Windows, Mac, Linux)
